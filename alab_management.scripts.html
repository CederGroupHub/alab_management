init_with_fake_parameters(cls: Type[alab_management.op_def.base_operation.BaseOperation]) → alab_management.op_def.base_operation.BaseOperation[source]¶

For task class, which need to be initialized to get the occupied positions and operation locations. We will initialized the task with a set of fake parameters based on their type annotations.

We implemented these faking types:

  • BaseDevice -> FakeDevice with name = {{name}}

  • SamplePosition -> SamplePosition with name = {{name}}

  • built-in types (can be nested) : int, float, str, list, dict, set

Note

Things like List[BaseDevice] and Dict[str, SamplePosition] are not supported up to now.

Parameters

cls – the operation class to be initialized
