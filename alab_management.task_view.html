class TaskStatus(value)[source]¶

Bases: enum.Enum

The status of one task

  • WAITING: the task cannot start now

  • READY: the task is ready to submit

  • RUNNING: the task is currently running

  • ERROR: the task encountered some errors during execution

  • COMPLETED: the task is completed

create_task(task_type, samples, parameters, prev_tasks=<alab_management.task_view.task_view._MISSING object>, next_tasks=<alab_management.task_view.task_view._MISSING object>)[source]¶

Insert a task into the task collection

Parameters

  • task_type (str) – the type of task, which should be a type name of class inherited from BaseTask

  • samples (Dict[str, ObjectId]) – the samples that this task will handle, which will be passed to Task object the same as parameters.

  • parameters (Dict[str, Any]) – the required tasks for this task

  • prev_tasks (Union[ObjectId, List[ObjectId]]) – one or a list of ObjectId that refer to prev tasks of this task (which must be completed before current task)

  • next_tasks (Union[ObjectId, List[ObjectId]]) – one or a list of ObjectId that refer to next tasks of this task (which cannot start until this task finishes)

ObjectId
